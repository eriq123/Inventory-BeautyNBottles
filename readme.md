# BeautyNBottles

## Installation

1. Download the zip file
2. Extract it to your http server 
3. Create a database named "beautynbottles_db"
4. Import the beautynbottles_db.sql file inside the database
5. Migrate and seed

```
php artisan migrate:fresh --seed
```

then

```
php artisan serve
```
